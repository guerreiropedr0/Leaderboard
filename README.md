# Template

> This repository is a template to all my projects.

This template includes Webpack, Stylelint, ESlint and Weblint.

## Built With

- HTML
- CSS
- JAVASCRIPT
- Node.js/NPM

## Getting Started

```
npm install
npm run build
npm start
```
